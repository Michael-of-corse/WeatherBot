# WeatherBot
## Телеграм бот для выдачи погоды
## Install
* Docker-container
* From source
### Docker-container
Кочаем репозиторий
`git clone https://github.com/michael-of-corse/WeatherBot.git`\
Запускаем контейнеры
```
docker compose build
docker compose up
```
### From source
Кочаем зависимости
`pip install -r requirments.txt`
Запускаем бота
`python3 start.py`
